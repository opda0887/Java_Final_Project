# Tutorial(使用教學)📝

### 🌀 - 執行方法 (Run)

在 VScode 的 terminal 中，輸入以下指令，來預覽遊戲畫面：

```
cd "absolute\path\to\Final_Project\" ; if ($?) { javac -encoding UTF-8 Game.java } ; if ($?) { java Game }
```

example: cd "C:\User\peter\Final_Project\" ; ...
